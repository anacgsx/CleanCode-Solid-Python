# Lista de Exercícios — Clean Code e SOLID com Python

Projeto desenvolvido para a disciplina **Arquitetura e Projeto de Software**, com foco na aplicação de boas práticas de desenvolvimento utilizando **Clean Code** e os princípios **SOLID** em Python.

---

## Objetivo

Este repositório reúne exercícios práticos voltados para:

* Escrita de código limpo e legível;
* Refatoração de código;
* Separação de responsabilidades;
* Redução de acoplamento;
* Aplicação dos princípios SOLID;
* Organização e modelagem orientada a objetos;
* Desenvolvimento de sistemas mais escaláveis e sustentáveis.

---

# Conceitos abordados

## Clean Code

Durante os exercícios foram aplicados conceitos como:

* Nomes significativos;
* Pequenas responsabilidades;
* Redução de duplicação;
* Organização do código;
* Uso de constantes;
* Métodos pequenos e objetivos;
* Melhor legibilidade;
* Separação de responsabilidades.

---

## SOLID

Os cinco princípios SOLID também foram aplicados:

| Princípio | Descrição                       |
| --------- | ------------------------------- |
| SRP       | Single Responsibility Principle |
| OCP       | Open/Closed Principle           |
| LSP       | Liskov Substitution Principle   |
| ISP       | Interface Segregation Principle |
| DIP       | Dependency Inversion Principle  |

# Como executar os exercícios

## 1. Clone o repositório

```bash
git clone https://github.com/anacgsx/CleanCode-Solid-Python.git
```

---

## 2. Acesse a pasta do projeto

```bash
cd CleanCode-Solid-Python
```

---

## 3. Execute um exercício

Exemplo:

```bash
python ex01.py
```
